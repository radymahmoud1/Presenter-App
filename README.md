**Presenter-App**
Presenter-App is an open-source desktop tool designed to enhance your presentations with powerful, easy-to-use on-screen utilities.

It brings together essential presentation features such as drawing, highlighting, zooming, and spotlighting — all in one lightweight application that works seamlessly in the background. It's recommended to disable it after ending your session to start typing the key letters on your keyboard. 

# Overview

PresenterApp is an open-source desktop application designed to enhance live presentations using professional on-screen presentation tools.

It combines essential presentation utilities such as spotlighting, highlighting, zooming, and visual pointing into a lightweight application that runs seamlessly in the background while presenting.

The application is especially useful for educators, researchers, lecturers, students, and professional presenters who require smooth audience interaction during presentations.

---

# Features

## 🖊️ Pen Pointer Tool
Display a clean on-screen pointer to guide audience attention during explanations and demonstrations.

## 🟡 Highlight Mode
Highlight important regions of the screen in real time to emphasize equations, figures, diagrams, or presentation content.

## 🔍 Zoom Integration
PresenterApp integrates with the built-in Windows Magnifier tool to provide smooth zoom functionality during presentations.

## 🔦 Spotlight Mode
Darken the surrounding screen while keeping a focused circular spotlight on the area of interest.

## 🎛️ Background Tray Mode
Runs silently in the background and can be accessed anytime from the Windows system tray / hidden icons area.

## ⌨️ Keyboard & Mouse Support
Fully controllable using standard keyboard and mouse input without requiring additional hardware.

## 🖥️ Multi-Monitor Support
Supports overlay rendering across multiple monitors and presentation displays.

## 🎮 Presenter Remote Compatibility
Compatible with presenter remotes such as:

- Baseus Orange Dot AI Wireless Presenter BS-OH085

---

# Controls

| Action | Result |
|---|---|
| Double press `E` | Cycle presentation tools |
| Triple press `E` | Clear active overlay tool |
| `Esc` | Disable current overlay |
