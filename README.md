# Presenter-App

Present Content with Presenter App that Includes Pen, Highlighter, Windows Zoom Feature, and Spotlight. 

The Zoom Features can be edited from Windows. 

Search on (magnifier) on the Windows search bar to find it and choose the best property that works well with your presentation style.

Setup the software provided (only on Windows) to start enjoying the presentation features addressed above. 

This App can work with your keyboard and mouse.

It works with the Baseus Orange Dot Ai Wireless Presenter BS-OH085 (make sure to connect its USB while turning the Bluetooth on)

The contributions to this Project might enhance capturing live transcription and recording using the Baseus Orange Dot features that became unsupported by the Baseus Company.
